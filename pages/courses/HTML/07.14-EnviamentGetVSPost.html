ANAR A L'INDEX

ENVIAMENT METODE GET VS POST

METODE GET

- És el mètode d'entrada a pàgines web quan nosaltres hi entrem a través d'una url

- Porta l'atribut action-> enviar.html

- Porta l'atribut method-> get

Sense atribut name

    
        <form action="enviar.html" method="get">
            <div>
                <label for="name">Nom</label>
                <input type="text" id="name">
            </div>
            <div>
                <label for="surname">Cognom</label>
                <input type="text" id="Surname">
            </div>
            <div>
                <input type="submit">
            </div>
        </form>

RESULTAT: Cannot GET /enviar.html

- Pel fet que si no s'insereix un document que existeixi, donarà error

- Malgrat posem el nom i cognom als camps, res no s'envia perquè en la url després del ? no hi ha cap informació

RESULTAT DE LA URL: http://127.0.0.1:5500/enviar.html?

Amb atribut name

    
        <form action="enviar.html" method="get">
            <div>
                <label for="name">Nom</label>
                <input type="text" id="name" name="name">
            </div>
            <div>
                <label for="surname">Cognom</label>
                <input type="text" id="Surname" name="surname">
            </div>
            <div>
                <input type="submit">
            </div>
        </form>

RESULTAT: Cannot GET /enviar.html

- Pel fet que si no s'insereix un document que existeixi, donarà error

- Ara, SI que s'envien dades al servidor, però com no hi ha cap document PHP anomenat enviar.html no es pot enviar

RESULTAT DE LA URL: http://127.0.0.1:5500/enviar.html?name=Asier&surname=P%C3%A1rraga

METODE POST

- Enviament de formulari per la porta de darrera.

- L'Error 405 que surt, significa que ha intentat enviar les dades, pero aquestes dades no han arribat al destí

- Porta l'atribut action-> enviar.html

- Porta l'atribut method-> post

- NO INFLUEIX EN RÉS que porti l'atribut name, però es una BONA PRAXIS

- És més segur que el get

Amb atribut name

    
        <form action="enviar.html" method="post">
            <div>
                <label for="name">Nom</label>
                <input type="text" id="name" name="name">
            </div>
            <div>
                <label for="surname">Cognom</label>
                <input type="text" id="Surname" name="surname">
            </div>
            <div>
                <input type="submit">
            </div>
        </form>

RESULTAT: HTTP ERROR 405

RESULTAT DE LA URL: http://127.0.0.1:5500/enviar.html

    --PASSOS PER VEURE DELS TEVES DADES--
  1. Inspecciona l'error
  2. Tira enrere
  3. Torna a enviar les dades
  4. Ves a Network
  5. Clica a enviar.html
  6. Ves a Payload
  7. Observa les teves dades

- Si no afegim name, el Payload no existirà
