Tipus text

- L'input per defecte

Atributs habituals

-Es fan servir els seguents atributs

  • name

- Serveix per dir-li al servidor amb quin tipus de dades estem treballant, si no s'esmenta aquest tipus de dades, el servidor no sap que fer

  • max-lenght

- Es útil per posar un màxim de caràcters que es podem posar a l'input

  • size

- Seveix per poder modificar la mida de la caixa de l'input

  • value

- Seveix per assignar un valor per defecte a l'input

  • placeholder

- Seveix per indicar a l'usuari quin valor s'espera que insereixi a l'input

  • pattern

- Va acompanyat d'una expressió regular o regex.

- Impedeix modificar el valor de l'input (gràcies a l'expressió regular)

- Es pot posar com [123456789] o [1-9]

- El símbol {} serveix per as4789signar quants valor hauràn d'estar entre aquest "rang" que assignem amb la regex.

Normalment NO DEIXA ENVIAR-HO, però JavaScript és més estricte, i NO DEIXA FER-HO

  • required

- És un valor booleà que fa que la casella s'hagi d'omplir SI O SI

- NORMA NO ESCRITA: per conveniència, es posa un asterisc (*) al label

Exemple de Google forms
