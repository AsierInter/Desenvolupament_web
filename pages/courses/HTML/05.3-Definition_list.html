TORNA A 05.Lists

<dl> [definition lists]

- Els elements han d'estar entre etiquetes dl significa 'definition list'

-Serveixen per definir diferents elements.

- Només accepten atributs globals (style, title, id, class )

Tipus d'etiquetes de les 'dl'

- Per emmagatzemar el contingut de la llista de definicions, posarem els 'items' entre les etiquetes dl i /dl

- La llista pot tenir diferents grups de termes, que es posaran entre les etiquetes dt i /dt

- I aquests grups de termes tenen diferents definicions que es posaran entre les etiquetes dd i /dd

Definicions

HTML
Llenguatge de programació frontend, que serveix per estructurar pàgines web.
CSS
Llenguatge de programació frontend, que serveix per editar visualment la 'plantilla' de la nostra web que hem fet previament amb HTML.
JavaScript
Llenguatge de programació backend, que serveix per estructurar la interactivitat amb la nostra pàgina web que previament hem editat visualment amb CSS i abans l'hem estructurat amb HTML.
