Estructura básica d'un formulari

- L'estructura básica d'un formulari d'HTML, está formada per les següents etiquetes:

  1. form
  2. label
  3. button
  4. input

Etiquetes

<form>

- Aquesta etiqueta englobarà tots els elements del formulari EXCEPTE els atributs globals

<label>

- Serveix per escriure el nom del camp a emplenar, es troba a dins de <form>

<input>

- Serveix perquè l'usuari pugui contestar al formulari, depenent del seu atribut es veurà d'una manera o una altra, hi han 22 tipus d'INPUT

<button>

Exemples


    <form>
        <label> Nom </label>
        <input>
        <button> Enviar formulari <button>
    </form>

- El comportament per defecte d'un formulari, es enviar les dades i recargar la pàgina

- Amb l'atribut action de l'etiqueta <form>, podem enviar les dades recullides a un document PHP, que es farà més endavant, perque es necesita un servidor.
