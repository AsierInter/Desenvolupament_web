INTRODUIR IMATGES WEBP

- La carpeta a la qual es recomana posar les imatges, és la carpeta assets

Imatges jpg, gif o png

Els dos atributs de img són els seguents:

  • src
  • alt

src

- Aquest atribut serveix per poder accedir a l'imatge des del servidor, si aquest text no està inserit, es mostrarà una imatge de "link trencat" i un text

alt

- Aquest atribut és el text que es mostra alhora de voler veure una imatge i que es mostri la imatge de "link trencat"

Exemple

    
        <img src="assets/dog.webp" alt="Imatge de gossos">

Imatges SVG

- Hi ha dues maneres d'inserir imatges SVG

  1. Amb els atributs de img
  2. Copiant el seu codi SVG

1. Amb els atributs de img

- Es fan servir l'atribut src i alt. A més, afegim els atributs height i width, per especificar les dimensions de l'imatge

    
        <img src="recursos_web/Github.webp" alt="Github_logo" height="50" width="50">
    

    
        <img src="recursos_web/unity-original.webp" alt="Unity_logo" height="50" width="50">

2. Copiant el codi SVG

- Hauràs d'accedir al seu codi mitjancant un editor de text

- Per inserir imatges d'aquesta manera, hem d'utilitzar l'etiqueta svg
