Taules

- Les taules serveixen per tabular informació.

- Els elements hauràn d'estar entre l'etiqueta table(que generarà la taula):

ETIQUETES

<tr> [table row]

- Construiex les cel·les

- El nombre de columnes estarà determinat pel nombre de cel·les posades a la primera etiqueta tr.

<td> [table data]

- Serveix per construir les cel·les i dins hi posem el contingut de les mateixes.
