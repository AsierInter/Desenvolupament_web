MÉS ELEMENTS DELS FORMULARIS

Etiqueta FIELDSET

- S'utilitza per agrupar elements dins d'un formulari

- S'utilitza l'etiqueta legend que permet posar un nom al contingut fieldset

    
        <form>
            <fieldset>
                <legend>Dades personals</legend>
                <div>
                    <label for="name">Nom</label>
                    <input type="text" id="name">
                </div>
                <div>
                    <label for="surname">Cognom</label>
                    <input type="text" id="surname">
                </div>
                <button>ENVIAR
            </fieldset>
        </form>

ETIQUETA METER

- A part de l'atribut indispensable value, porta els atributs min, max, low, high i optimum

    
        <form>
            <label for="fuel">Combustible</label>
            <meter id="fuel" min="0" max="100" value="90" low="0" high="75" optimum="10"></meter>
        </form>

- Si el value es troba més a prop de l'optimim, estarà en verd

- Si el value està molt per sota de l'optimum, estarà en groc

ETIQUETA PROGRESS

- Crea una barra de progrés en funció del seus atributs: value i max

- Per molt que cambiem el valor de max, la barra no es farà més gran o més petita

-Una BONA PRÀCTICA és treballar amb percentatges

    
        <form>
            <label for="task">Tasca 1</label>
            <progress id="task" max="100" value="90"> </progress>
        </form>
