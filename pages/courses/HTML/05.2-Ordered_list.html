TORNA A 05.Lists

<ol> [ordered lists]

- Els elements han d'estar entre etiquetes <ol> significa 'ordered list'.

- Els elements han d'estar entre etiquetes <li>, dins de la <ol>.

Atribut type

- El type de les ol, assignen diferents valor per després editar amb CSS.

    'a' indica lletres en minuscula per ordenar
  1. U
  2. DOS
  3. TRES
  4. CUATRE
    'A' indica lletres mayúscules per ordenar
  1. U
  2. DOS
  3. TRES
  4. CUATRE
    'i' indica nombres romans en minuscula per ordenar
  1. U
  2. DOS
  3. TRES
  4. CUATRE
    'I' indica nombres romans en mayúscules per ordenar
  1. U
  2. DOS
  3. TRES
  4. CUATRE
    '1' indica nombres per ordenar
  1. U
  2. DOS
  3. TRES
  4. CUATRE

Atribut start

-Si li afegim l'atribut start, podrem dir des d'on començarà la nostra llista

  1. U
  2. DOS
  3. TRES
  4. CUATRE

Llistes imbrincades

Dins d'una <ol> podem posar elements <li>, pero per imbrincarles fiquan altres llistes <ol> a dins.

  1. Element 1
  2. Element 2
  3. Element 3
    1. Element imbrincat 1
    2. Element imbrincat 2
    3. Element imbrincat 3
