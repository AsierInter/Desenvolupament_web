ETIQUETA META

Etiqueta META NAME VIEWPORT CONTENT

- Serveix perquè la teva pàgina web s'ajusti a qualsevol dispositiu de cerca

    
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

Etiqueta META NAME CHARSET

- Atribut necesari per poder visualitzar correctament tots els caracters de les llengues europees pel fet que la codificació és l'UTF-8. Serveix per la codificació dels caràcters

    
            <meta charset="UTF-8">

Etiqueta META NAME DESCRIPTION

    
        <meta name="description" content="Lorem ipsum">

- Porta un text com a contingut

- Aquest és el text que apareix a sota de l'enllaç del link d'interés

Etiqueta META NAME AUTHOR

    
        <meta name="author" content="asierparraga">

- Aquesta etiqueta ens explica qui és el propietari de la pàgina web
