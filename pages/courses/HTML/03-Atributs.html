03-Atributs

- El valor de l'atribut, no pot contenir espais, ja que si no,CSS no acceptarà aquesta tribut odonarà error

- L'atribut més important és l'ID (i és únic) i amb CSS podrem treballar amb més facilitat amb elements "destacables"

- Un dels atributs més importants es class, que serveix per donar estils amb el fitxer CSS

- El "tooltip" de title, serveix per ajudar-nos a entendre les funcions dels diferents elements d'HTML

- data-* (serveix per emmagatzeman dades en un element html)

Tipus d'atributs
