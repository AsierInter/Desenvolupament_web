Conectar HTML i CSS

- Les cuatre maneres per conectar html i css són les seguents:

  1. Document style assignat
  2. Utilitzar l'etiqueta style al head
  3. Utilitzar l'atribut style als elements html
  4. Utilitzar l'etiqueta import

1.Document style assignat

- Linkem al head de cada document al seu respectiu document CSS, que estarà dins d'una carpeta a la mateixa del document

    
            <link rel="stylesheet" href="./css/styles.css">

2. Etiqueta <style> dins del head

- Dins de l'etiqueta style tindrem tot el nostre codi CSS

- Si ho fessim així, TOTS els nostres html haurien de tenir aquest codi CSS

- Així i tot, es recomana tenir coherencia entre els arxius de la nostra web, per així no confondre a cap integrant de l'equip o a nosaltres mateixos.

    
        <head>
            <style>
                body {
                    background-color: color-mix(in srgb, red 20%, black 40%)
                }
            </style>
        </head>

3. Utilitzar l'atribut style als elements html

- Això és una MALA PRAXIS, no es recomana gens fer-ho

- Als diferents elements html podem donar-li estils

- Només es pot utilitzar quan tenim JavaScript per ajudar-nos a editar el nostre document

    
        <h2 style="color: red;">MALA PRAXIS

4. Utilitzar la regla @import

- Afegim una regla import a l'etiqueta style

- Això en html equival al link de la primera opció

- No es recomana gens alhora de fer la nostra pàgina web, pel fet que els documents html i CSS carreguen de manera asincrona, i si afegim la regla import, al principi trigarà a carregar una mica i després es mostraran els estils (dona molt mala impatge a la nostra pàgina web)

    
        <head>
            <style>
                @import url(css/styles.css);
            </style>
        </head>
