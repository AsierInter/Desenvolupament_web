Sintaxis de CSS

Selector d'etiqueta

- És un element al que li aplicarem estils. Hi ha diferents tipus de selectors. Obrim i tanquem claus alhora de colocar-ho

    
        selector{

        }

Propietats de selector

- No hi ha limit de propietats que podem assignar a un element

    
        selector{
        propietat 1:
        propietat 2:
        propietat 3:
        }

Valor de la propietat

- És el paràmetre que agafa aquella propietat que volem modificar

PROPIETAT + VALOR = DECLARACIÓ

- Entre el valor i la següent propitetat hi va un punt i coma (;)

    
        selector{
        propietat 1:valor;
        propietat 2:valor;
        propietat 3:valor;
        }

RESULTAT:

    
        selector{
        declaració 1
        declaració 2
        declaració 3
        }

Bloc de declaració

- Cada bloc de declaració, és el conjunt de declaracions que modifiquen el nostre selector 

    
        selector{
        bloc;
        de;
        declaració;
        }

REGLA DE CSS

- És el conjunt de

    
        selector{
        declaració 1
        declaració 2
        declaració 3
        }
