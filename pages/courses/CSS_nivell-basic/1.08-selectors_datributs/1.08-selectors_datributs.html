Selectors d'atributs en CSS

  1. Selector d'ID

    2. - Haurem d'asignar un id a qualsevol etiqueta HTML (p,h1,li,ol...)

    - Per donar-li un atribut en CSS, hem d'utilitzar #

            
            <h2 id="second_title">Selector d'ID</h2>
            #second_title {
            background-color: red;
            }

    - És aconsellable NO posar els ID amb mayúscules, o almenys si es posen en mayúscules, SEMPRE haurà de ser així

    - Així i tot, MAI hem d'utilitzar els ID per donar estils

  2. Selector de classe

    3. - Haurem d'asignar un class a qualsevol etiqueta HTML (p,h1,li,ol...)

    - Per donar-li un atribut en CSS, hem d'utilitzar .

                
                <h2 id="third_title class=" third_title">Selector de classe</h2>
                .third_title {
                background-color: goldenrod;
                }

    - És recomenable utilitzar la METODOLOGIA BEM per utilitzar els selectors de les classes i també la METODOLOGIA DE LES CASCADES I L'HERÈNCIA
